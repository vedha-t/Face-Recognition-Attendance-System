# Face Recognition Attendance System

## About Project
A Face Recognition based Attendance System that automatically detects and recognizes faces using computer vision techniques and marks attendance.

## Technologies Used
- Python
- OpenCV
- Face Recognition
- Machine Learning
- MySQL

## Features
- Face image capturing
- Face model training
- Real-time face recognition
- Automatic attendance marking

## Project Files
- capture_faces.py - Capture user face images
- train_model.py - Train face recognition model
- recognize_face.py - Recognize faces and mark attendance
- face_detection.py - Face detection module
